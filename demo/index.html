Mobile & User friendly

Vanilla Form is not only responsive and optimized for hi-resolution devices (like Retina Displays). To improve user experience of your web page Vanilla Form turns on adequate virtual keyboard on mobile devices.

Ready to use anywhere you want

Vanilla Form uses JavaScript to send filled data to server using AJAX. This means, your page won't reload after form submit. Vanilla Form written with speed and independence in mind. No 3rd party library is needed like jQuery. But if you want to, you can easily integrate it with any web framework you want.

Anti-spam protection

Hate SPAM and captcha? So do I. Vanilla Form has an no annoying captcha. Thanks for smart built-in anti-bot protection it saves you from most of unwanted messages.
